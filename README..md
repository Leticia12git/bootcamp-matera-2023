# CARTEIRA DIGITAL - MATERA

## Sumário


### 1. Tecnologias Utilizadas

- Java (JDK 17)
- Spring Boot (Running with Spring Boot v2.7.5, Spring v5.3.23)
- JPA
- Hibernate
- Lombok
- Model Mapper
- Mysql (MySQL 5.7)

* [Maven](https://maven.apache.org/download.cgi);
